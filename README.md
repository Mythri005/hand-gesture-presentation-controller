# Hand Gesture Based Presentation Controller

A real-time computer vision application that enables **touchless control of presentation slides** using hand gestures through a webcam.  
Compatible with **PowerPoint, WPS Presentation, and other slideshow tools**.

## Features
- Real-time hand tracking via webcam
- Swipe gestures for slide navigation
- Fist gesture to pause/resume slides
- Fast and smooth gesture response
- Works across multiple presentation software

## Gesture Controls

| Gesture | Action |
|--------|--------|
| Swipe Right | Next Slide |
| Swipe Left | Previous Slide |
| Fist | Pause / Resume |

## Tech Stack
Python, OpenCV, MediaPipe, PyAutoGUI, NumPy
